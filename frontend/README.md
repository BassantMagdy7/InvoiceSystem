# Invoice System Frontend

Angular frontend for the Invoice System application.

## Features

- User login and signup pages
- Authentication using HttpOnly cookie from the backend
- Protected invoice dashboard
- Create, edit and delete invoices
- Add multiple invoice items
- Store, customer and product dropdowns loaded from backend APIs
- Automatic item total and invoice total calculation
- Invoice list with pagination
- Search/filter invoice list
- SignalR real-time invoice list refresh
- Logout functionality
- Responsive UI using HTML and CSS

## Technologies

- Angular
- TypeScript
- HTML
- CSS
- SignalR client
